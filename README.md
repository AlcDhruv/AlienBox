## What is this?

This is a small 2D action platformer I'm building in **Godot** while learning the engine.
The idea is simple: move around, jump (yes, there’s a double jump), and interact with the environment — including some weird floating nails that only react when your character’s aura touches them.

I'm making everything myself:

* Code
* Art
* Game logic
* Probably some questionable design decisions

So expect things to break, get redesigned, and break again.

## Current Features

* Character movement
* Double jump
* Aura system
* Physics interactions with nearby objects
* Animated sprites

## Why I'm making this

Partly to learn **Godot**, partly to challenge myself to build a full game solo — code, art, and design.

Also because making weird mechanics is fun.
